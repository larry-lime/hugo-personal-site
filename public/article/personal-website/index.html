Getting Started#

  1. Install Code Editor: VScode

  2. Install Package Manager: Homebrew

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  3. Install Static Site Generator Hugo

    brew install hugo
  • Follow the directions on the terminal output

Create your Website#

  1. Run the following commands in your terminal

  2. Create Hugo Project

    hugo new site quickstart -f yml

  3. Install Theme

    cd quickstart
git init
git submodule add https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod

  4. Open the project in VSCode

    open . -a Vscode.app

  5. Copy the following code into your config.yml
